Tips-Tricks

There may be various reasons for an ISP(Internet Service Provider) to block certain websites
Most reasons may be to follow certain protocols or rules

1.Some websites are blocked only by thier domain name and can easily be accessed by thier IP address.
To find out IP address of any website, type in "ping yourdomain.com" in command prompt on your Windows OS (note:here yourdomain=the website you want to access).
Now type the IP address recieved into the URL of your browser and access the website.

2.Proxy websites can be used to access any websites.
Go to any proxy website and enter the name of the website you want to access.
Example:proxysite.com

3.Some blocked websites can be access just by accessing the secured versions of thier websites
What can be done is that instead of "http://yourdomain.com" we cant use "https://yourdomain.com" ,
this will allow us to access the secured version of the websites.
Example:"https://kat.cr"
4.Websites can also be accessed using a VPN service
Any VPN software can be used to access a website
A Virtual Private Network or VPN is a remote server that gives you private access through a public network.
VPN makes it looks as though you are browsing from the country where the server is located, thereby hiding your real identity.
To access any website using VPN, simply open the software and click on any click on any of the country names on the list and access internet through that server.
Example:Tunnelbear VPN.
